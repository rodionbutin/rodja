После введения в строку браузера адреса сайта и нажатии кнопки "Enter" отправляется запрос в файл Hosts на жестком диске для соотношения адреса сайта в тектовом формате с ip адресом. Если данных в файле hosts нет, то тогда запрос отправляется на DNS-сервер для получения ip адреса указанного веб сайта. Когда ip адрес получен, на него отправляется запрос. после получения этого запроса, отправлется ответ, это и есть сайт.
